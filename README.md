# React Fluency Demos

A collection of interactive demos showcasing advanced UI patterns and animation fluency in React. This project demonstrates best practices for transitions, anticipation, continuity, control, and intention using modern React, Framer Motion, and Ant Design.

## Features

- ✨ Modern React (v19) with functional components and hooks
- ⚡ Fast development with Vite
- 🎬 Smooth animations using Framer Motion
- 🧩 Modular demo structure for easy extension
- 🎨 Styled with Ant Design and custom tokens
- 🗂️ Organized codebase for learning and experimentation

## Demo Structure

- **Anticipation**: Prepares users for upcoming changes with subtle cues.
- **Continuity**: Maintains context and flow during UI transitions.
- **Control**: Demonstrates user-driven state changes and feedback.
- **Intention**: Optimistic UI updates and server interaction patterns.

## Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```sh
   git clone https://git.svb.lacaixa.es/U01AA426/react-fluency-demos.git
   cd react-fluency-demos
   ```
2. Install dependencies:
   ```sh
   npm install
   # or
   yarn install
   ```

### Running the Project

To start the development server:

```sh
npm run dev
# or
yarn dev
```

Open your browser at [http://localhost:5173](http://localhost:5173) (or the port shown in your terminal).

### Building for Production

```sh
npm run build
# or
yarn build
```

### Preview Production Build

```sh
npm run preview
# or
yarn preview
```

## Folder Structure

- `src/demos/` — Demo modules for each UI pattern
- `src/components/` — Shared components
- `src/styles/` — Styling tokens and shared styles
- `src/constants/` — App-wide constants and routes
